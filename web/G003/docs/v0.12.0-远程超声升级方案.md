# G003 超声RIS 远程超声模块升级方案 v0.12.0

## 一、需求背景

主公指示：对标国内前五远程超声厂商（麦迪克斯、蓝网、华大智造、华声、莱塞思），将相关技术/功能/模块/参数补充到G003超声RIS。

## 二、联合搜索结论

| 厂商 | 核心产品 | 技术特点 |
|------|---------|---------|
| **华声医疗** | "四叶草" 5G远程超声 | 内置通讯模块，零延时零失真，反向控制，云端存储，2000+用户，20000+会诊 |
| **华大智造** | "云影睿诊" | 多路影像实时同步（超声图像+操作手法），5G机器人，移动车 |
| **蓝网科技** | 远程超声会诊平台 | B/S架构，音视频+数据三位一体，区域医联体 |
| **麦迪克斯** | 超声科信息化系统 | 全流程信息化管理 |
| **莱塞思** | 远程超声诊断方案 | 超声图像+操作手势同步采集，语音/视频/标注 |

**功能清单**：
- 实时超声图像传输 + 操作手法同步
- 反向控制（专家远程调节超声参数）
- 多路视频会诊（申请方/会诊方/患者/超声图像）
- 云端数据存储与回放
- 5G网络质量监控（带宽/延迟/抖动/丢包）
- 结构化会诊报告（诊断/建议/随访/危急值）
- 自动降质切换（高清→标清→音频）
- 医联体协作网络（在线专家状态）

## 三、升级方案

**对现有RemoteConsultationPage（279行）做保守扩充，不重写UI框架。**

### 升级内容

#### 1. 扩充会诊申请Tab
- 状态筛选（全部/待会诊/会诊中/已完成）
- 新增"新建申请"按钮
- 右侧详情增加：检查类型、下级医院、紧急度

#### 2. 新增"实时会诊"Tab
- 四分屏：超声图像 / 操作手法 / 申请方专家 / 会诊方专家
- 超声参数显示区（深度/增益/频率/TGC）
- 底部标注工具栏（箭头/画笔/文字/清除）
- 右侧信息栏（患者/带宽/延迟/通话计时）
- 数据：2条进行中实时会诊

#### 3. 新增"会诊报告"Tab
- 会诊选择器（已完成列表）
- 结构化报告：诊断结论/诊疗建议/随访建议/危急值标注
- 会诊满意度（5星评分）
- 申请医生/会诊专家签名输入
- 保存/打印按钮
- 数据：3条示例报告

#### 4. 扩充图文交流Tab
- 消息时间戳
- 支持文字/图片/语音三种消息类型
- 实时网络状态显示

#### 5. 扩充医联体协作Tab
- 左侧：区域医院网络（4家虚构医院）
- 右侧：5G网络质量监控（带宽/延迟/抖动/丢包，进度条+颜色）

## 四、数据安全

虚构医院名：东华区第一医院、南港新城中心医院、国家医学中心直属医院、滨海新城医院
虚构患者/医生：全部使用虚构姓名

## 五、技术约束

- 配色：#1a3a5d（主色），侧边栏白色字体20px
- 内联样式（非Tailwind）
- React + TypeScript（移除@ts-nocheck）
- 保持现有3 Tab结构，新增2个Tab，共5个Tab
- 不使用真实厂商名称作为UI标签

## 六、验收标准

- TypeScript 0 errors
- Playwright 13路由全量验收（blank=false, errors=0）
- /remote-consultation 5个Tab全部可点击切换
- 0白屏0JS错误
