# 放射RIS系统竞品调研报告

> 调研时间：2026年5月1日
> 调研对象：蓝网科技、东软医疗、联影医疗、GE医疗、锐科医疗、岱嘉医学

---

## 一、蓝网科技放射RIS / 医学影像系统

### 1. 产品定位
蓝网科技是国内专注于医学影像信息化领域的厂商，其放射RIS系统主要面向二级及以上医院的放射科，提供完整的放射检查工作流和报告管理，同时具备与PACS深度融合的能力。

### 2. 整体架构
- **架构类型**：B/S架构，基于Web的放射科室管理系统
- **部署方式**：本地化部署，支持私有云
- **数据库**：支持 Oracle / SQL Server / MySQL
- **接口标准**：DICOM MWL、HL7 v2.x、DICOM 3.0
- **技术特点**：模块化设计，支持负载均衡集群部署

### 3. 核心功能模块

| 模块 | 功能描述 |
|------|---------|
| **检查预约** | 放射检查自助预约，支持分时段预约管理 |
| **工作列表** | DICOM MWL集成，实时检查进度跟踪，叫号分诊 |
| **报告书写** | 结构化报告模板，支持语音输入，多级审核 |
| **图像管理** | 放射影像采集、存储、调阅，支持CR/DR/CT/MR等 |
| **PACS集成** | 与各类PACS系统对接，支持影像归档与调阅 |
| **统计分析** | 检查量统计、工作量统计、收入分析报表 |
| **质量管理** | 报告质控、危急值管理、修改痕迹保留 |
| **设备管理** | 放射设备状态监控、预约与使用管理 |

### 4. 特色功能 / 亮点
- 放射科专用分诊叫号系统，与检查流程深度集成
- 多种检查类型（DR/CT/MR/乳腺）的标准化分类管理
- 影像与报告一体化展示，支持双屏工作模式
- 放射特有"检查部位-检查方法"标准化字典
- 支持超声+放射双模式（部分版本）

### 5. 适用场景
- 二级及以上医院放射科
- 超声+放射一体化科室
- 需要私有化部署的医疗机构

## 二、东软医疗 Neusoft 放射RIS系统

### 1. 产品定位
东软医疗的PACS/RIS是其智慧医学影像信息系统的核心组成部分，面向大型三级医院，支持放射、超声、内窥镜、病理、电生理、核医学、介入等多科室，是国内医疗信息化领域产品线最完整的厂商之一。

### 2. 整体架构
- **架构类型**：B/S架构，企业级云化架构
- **部署方式**：支持本地化、私有云、公有云混合部署
- **数据库**：支持大型数据库集群（Oracle RAC）
- **标准支持**：完全符合 DICOM 3.0、HL7 v2.x 及 IHE 规范
- **技术栈**：负载均衡、集群部署，全院影像数据集中存储，支持微服务化

### 3. 核心功能模块

| 模块 | 功能描述 |
|------|---------|
| **影像存储与调阅** | 影像数据永久安全管理，负载均衡面向全院提供调阅及后处理服务 |
| **专科诊断工作站** | 支持放射、超声、内窥镜、病理、电生理、核医学、介入等多科室 |
| **业务流程信息化** | 检查预约、登记、检查、执行、报告全流程闭环 |
| **智能辅助诊断** | AI辅助诊断集成，基于疾病诊断知识库 |
| **结构化报告** | 基于疾病诊断知识库的结构化报告，支持模板定制 |
| **危急值管理** | 检查闭环和危急值闭环完全管理，双闭环设计 |
| **临床系统交互** | 与HIS/EMR/电子病历系统深度集成 |
| **绩效管理** | 全面的业务、质量、绩效管理，多维度绩效考核 |
| **质控管理** | 切实加强影像质量安全管控 |
| **教科研支持** | 支撑影像教科研发展 |
| **N-VNA数据平台** | 智慧影像数据平台，统一管理全院影像数据 |

### 4. 特色功能 / 亮点
- **多科室统一平台**：放射+超声+病理等多模态统一管理
- **检查闭环+危急值闭环**：双闭环设计，完全符合互联互通、电子病历等评级要求
- **N-VNA智慧影像数据平台**：统一影像数据管理，支持跨科室调阅
- **AI辅助诊断集成**：基于疾病诊断知识库的结构化报告
- **全面评级解决方案**：提供互联互通、电子病历等完整评级支持
- **负载均衡技术**：保障全院级大规模并发访问
- **智慧物联+AI+大数据**：新一代信息技术整合
- **云平台架构**：支持混合云部署

### 5. 适用场景
- 大型三甲医院全院影像信息化
- 需要通过互联互通/电子病历评级的医院
- 多院区/医疗集团统一影像管理
- 医联体/区域影像数据中心

## 三、联影医疗 (uCT) 放射RIS系统

### 1. 产品定位
联影医疗是国内高端医学影像设备供应商，其信息管理系统（UIHPACS）面向大型三甲医院，提供从设备到信息化整体解决方案，设备与信息系统深度融合是最大特色。

### 2. 整体架构
- **架构类型**：B/S架构，云计算架构
- **部署方式**：支持院内私有云、区域云平台部署
- **数据库**：大型分布式数据库
- **标准支持**：DICOM 3.0、HL7 v2.x、IHE、DICOM SR结构化报告

### 3. 核心功能模块

| 模块 | 功能描述 |
|------|---------|
| **检查工作流** | DICOM MWL集成，自动获取检查申请 |
| **报告管理** | 结构化报告书写，支持图文一体化报告 |
| **影像管理** | 全系列设备（CT/MR/XR/PET-CT）影像统一管理 |
| **PACS集成** | 与联影CT/MR/XR等设备无缝对接 |
| **设备管理** | 放射设备状态监控、预约管理 |
| **统计分析** | 科室工作量、设备使用率统计 |
| **危急值推送** | 与临床系统实时联动 |
| **区域互联** | 支持医联体/区域影像数据中心 |
| **AI辅助诊断** | 与联影AI平台（uAI）无缝集成 |

### 4. 特色功能 / 亮点
- **高端设备配套**：与联影CT/MR/XR等设备深度原生集成
- **云PACS架构**：支持多院区、区域影像数据中心
- **DICOM SR结构化报告**：兼容国际标准的结构化报告
- **统一认证**：与医院一卡通/人脸识别系统集成
- **uAI平台AI辅助**：与联影AI平台无缝集成
- **区域互联/医联体**：支持跨机构影像数据共享

### 5. 适用场景
- 大型三甲医院放射科（尤其是使用联影设备的医院）
- 多院区医院统一影像管理
- 区域医联体/影像数据中心
- 需要区域影像共享的卫生体系

## 四、GE医疗 放射RIS / Centricity影像系统

### 1. 产品定位
GE医疗是全球领先的医学影像设备及信息化提供商，其Centricity RIS是面向全球大型医院的企业级放射信息系统，与GE的高端影像设备（CT/MR/XR/超声）深度集成，主打企业级云化和AI驱动的智能化影像管理。

### 2. 整体架构
- **架构类型**：B/S架构，企业级云化架构（Centricity Cloud)
- **部署方式**：支持本地私有云、公有云、混合云部署
- **数据库**：支持Oracle/SQL Server大型数据库
- **标准支持**：DICOM 3.0、HL7 v2.x/IHE、ISO-IR/IHE RDV
- **技术栈**：Enterprise Imaging（企业级影像）平台，AI驱动的影像工作流

### 3. 核心功能模块

| 模块 | 功能描述 |
|------|---------|
| **影像存储与调阅** | Enterprise Imaging (VNA) 统一影像管理，支持全院级调阅 |
| **RIS工作流** | 放射检查预约、登记、分诊、报告全流程管理 |
| **诊断工作站** | 高性能诊断阅片，支持多屏/3D后处理 |
| **报告管理** | 结构化报告，支持语音录入 |
| **AI集成** | 与GE AI产品线（CareIntellect等）深度集成 |
| **设备集成** | 与GE CT/MR/XR/超声设备原生对接 |
| **企业级管理** | 多站点、多院区统一管理，角色权限体系 |
| **数据洞察** | Imaging 360数据分析和运营报表 |
| **网络安全** | Centricity Cybersecurity企业级安全认证 |
| **云部署** | Centricity Cloud云平台支持 |

### 4. 特色功能 / 亮点
- **Enterprise Imaging（企业影像）理念**：将所有影像（放射/超声/心脏病学等）统一到单一平台
- **Centricity Cybersecurity**：企业级安全认证，适合大型医疗机构
- **AI Orchestration（AI编排）**：与GE的AI平台无缝集成，实现AI辅助诊断
- **Imaging 360**：统一的影像运营数据分析和绩效管理
- **与GE设备原生集成**：软硬件一体化，设备参数自动获取
- **全球案例丰富**：在全球大型医院有大量部署案例
- **DICOM SR支持**：完整的结构化报告标准支持

### 5. 适用场景
- 大型三甲医院，特别是使用GE影像设备的医院
- 外资/合资医院
- 多院区/医疗集团统一影像管理
- 需要通过JCI等国际认证的医院
- 对数据安全和合规性要求极高的医疗机构

### 7.2 优缺点总结

| 厂商 | 核心优点 | 主要缺点 |
|------|---------|---------|
| **蓝网科技** | 叫号分诊集成好、超声+放射双覆盖、成本适中 | 无AI、无云架构、仅支持私有云 |
| **东软医疗** | 双闭环、N-VNA、AI+知识库、全面评级支持、多科室统一平台 | 价格较高、体系庞大、实施复杂 |
| **联影医疗** | 设备+信息化深度融合、云PACS架构、uAI平台、区域互联 | 主要适合已用联影设备的医院 |
| **GE医疗** | Enterprise Imaging全球领先、AI编排、Centricity安全认证、与GE设备原生集成 | 价格最高、本地化服务能力弱 |
| **锐科医疗** | 轻量化高性价比、模块化灵活、实施周期短 | 无AI、无云架构、功能相对基础 |
| **岱嘉医学** | 危急值全生命周期、结构化报告引擎灵活、评分体系完善 | 规模较小、无AI和云架构 |
