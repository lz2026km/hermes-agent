## 1. 系统概述

### 1.1 系统简介

G002 院区核医学信息管理系统（Nuclear Medicine Information System，简称NMIS）是一套专为核医学科设计的综合性临床信息管理平台。系统涵盖患者管理、检查预约、图像采集、报告书写、药品管理、辐射防护、质量控制等核医学科核心业务，实现了核医学诊疗全流程的数字化、规范化管理。

本系统对标国内领先的麦迪克斯 MEMRS-NM-B4.0 分子影像与核医学信息管理系统，适用于三级甲等医院核医学科的日常运营管理。

### 1.2 主要功能模块

系统包含7大子系统，共计15个功能页面：

| 序号 | 子系统名称 | 功能页面数 | 主要功能 |
|------|------------|------------|----------|
| 1 | 患者管理子系统 | 1个 | 患者档案管理、辐射风险评估 |
| 2 | 预约管理子系统 | 2个 | 检查预约、排班管理 |
| 3 | 检查管理子系统 | 2个 | 工作列表、检查记录管理 |
| 4 | 报告管理子系统 | 2个 | 报告查询、报告书写 |
| 5 | 药品管理子系统 | 2个 | 药品库存、药品分发 |
| 6 | 辐射防护子系统 | 1个 | 个人剂量监测 |
| 7 | 质量控制与统计子系统 | 4个 | 质控管理、统计报表、培训管理、系统管理 |

### 1.3 系统特点

#### 1.3.1 全流程覆盖
从患者预约、检查执行、药品使用到报告发布，系统覆盖核医学诊疗全流程，各环节数据互通互联，告别信息孤岛。

#### 1.3.2 实时数据监控
首页概览实时展示今日检查数量、药品库存、辐射剂量、设备状态等关键指标，科室运营状态一目了然。

#### 1.3.3 智能预警机制
系统具备药品效期预警、库存不足预警、辐射剂量超阈值预警、设备质控预警等多维度预警功能，提前发现风险点。

#### 1.3.4 规范化诊疗流程
系统内置标准化的检查项目和报告模板，支持自定义模板配置，保证诊疗行为的规范化、标准化。

#### 1.3.5 放射性药品全生命周期管理
从药品入库、存储、分发到废弃处理，实现放射性药品的全生命周期追溯管理，符合国家放射性药品管理法规要求。

### 1.4 适用检查类型

系统支持以下核医学检查类型的全流程管理：

| 检查类型 | 典型项目 | 代表性药品 |
|----------|----------|------------|
| PET/CT | 全身肿瘤筛查、肿瘤分期、疗效评估 | 18F-FDG、18F-PSMA、18F-AV45 |
| SPECT | 骨显像、心肌灌注显像、肾动态显像、甲状腺显像 | 99mTc-MDP、99mTc-MIBI、99mTc-DTPA |
| 核素治疗 | 甲亢碘-131治疗、骨转移瘤89Sr治疗 | 131I-NaI、89SrCl2 |
| 骨密度 | DXA骨密度测定 | 无放射性药物 |

### 1.5 硬件设备支持

系统支持与以下核医学设备的数据对接：

- PET/CT设备（Biograph Vision 600等）
- SPECT设备（Symbia Intevo Bold等）
- 骨密度仪（DXA系列）
- 剂量仪（放射性药品活度计）
- 个人剂量计

## 3. 7大子系统15个功能页面详细介绍

### 子系统一：患者管理子系统

#### 3.1 患者管理页面（/patients）

**页面路径**：首页 → 患者管理

**页面功能**：患者档案管理是核医学科信息化管理的基础环节。系统建立完整的患者电子健康档案，记录患者基本信息、联系方式、过敏史、辐射风险等级等关键数据，为后续检查预约、报告书写提供数据支撑。

**主要功能**：

1. **患者档案查询**
   - 支持按姓名、身份证号、手机号、PET/CT号等多条件查询
   - 搜索结果实时显示，快速定位目标患者

2. **患者档案展示**
   - 基本信息：姓名、性别、年龄、身份证号
   - 联系方式：手机号、家庭住址
   - 医学信息：过敏史、辐射风险等级
   - 就诊信息：PET/CT号、末次就诊日期

3. **辐射风险评估**
   - 系统根据患者检查历史和当前检查项目自动评估辐射风险等级
   - 风险等级分为：低风险、中风险、高风险
   - 高风险患者在检查流程中会显示特殊警示标识

**数据字段说明**：

| 字段名 | 说明 | 示例 |
|--------|------|------|
| id | 患者唯一标识 | P001 |
| name | 患者姓名 | 李建国 |
| gender | 性别 | 男/女 |
| age | 年龄 | 62 |
| phone | 联系电话 | 13800001001 |
| idCard | 身份证号 | 310101196204153317 |
| address | 家庭住址 | 上海市静安区 |
| radiationRisk | 辐射风险等级 | 低/中/高 |
| petctNo | PET/CT编号 | PET20260501001 |
| lastVisit | 最后就诊日期 | 2026-05-01 |

**操作流程**：
1. 在搜索框输入患者姓名或身份证号
2. 点击查询按钮或按回车键
3. 在搜索结果中找到目标患者
4. 点击患者姓名查看详细信息

### 子系统二：预约管理子系统

#### 3.2 预约管理页面（/appointments）

**页面路径**：首页 → 预约管理

**页面功能**：预约管理是核医学科高效运转的核心环节。系统提供检查项目预约功能，支持PET/CT、SPECT、核素治疗、骨密度等多种检查类型的预约登记，合理安排检查时间，优化患者就诊体验。

**主要功能**：

1. **预约登记**
   - 选择患者（从已有患者库选择或新建患者）
   - 选择检查类型：PET/CT、SPECT、核素治疗、骨密度
   - 选择检查项目：全身肿瘤筛查、肿瘤分期、疗效评估等
   - 设定预约日期和时间
   - 选择主诊医生
   - 备注特殊要求（如糖尿病史、禁食要求等）

2. **预约状态管理**
   - 待确认：新建预约，等待确认
   - 已确认：预约已确认，等待检查
   - 检查中：患者正在检查
   - 已完成：检查已完成
   - 已取消：预约已取消

3. **药品预分配**
   - 预约时可预分配放射性药品
   - 显示药品名称、剂量
   - 支持备注特殊准备要求

**数据字段说明**：

| 字段名 | 说明 | 示例 |
|--------|------|------|
| id | 预约唯一标识 | APT001 |
| patientId | 患者ID | P001 |
| patientName | 患者姓名 | 李建国 |
| examType | 检查类型 | PET/CT |
| examItem | 检查项目 | 全身肿瘤筛查 |
| scheduledDate | 预约日期 | 2026-05-02 |
| scheduledTime | 预约时间 | 08:30 |
| status | 预约状态 | 已完成 |
| radiopharmaceutical | 放射性药物 | 18F-FDG |
| injectedDose | 注射剂量 | 5.5mCi |
| doctor | 主诊医生 | 张明华 |
| notes | 备注 | 糖尿病史 |

**预约操作流程**：
1. 点击"新建预约"按钮
2. 填写或选择患者信息
3. 选择检查类型和具体项目
4. 设定预约日期和时间
5. 选择主诊医生
6. 如有特殊要求，在备注中说明
7. 点击"保存"完成预约

#### 3.3 排班管理页面（/schedule）

**页面路径**：首页 → 排班管理

**页面功能**：排班管理用于管理核医学科医护人员的工作排班，确保科室24小时运转有序。系统支持查看和管理医生、护士、技师、物理师的值班安排。

**主要功能**：

1. **排班视图**
   - 按周/月视图展示排班信息
   - 清晰显示每位人员的值班日期和时间段

2. **人员信息展示**
   - 姓名、科室、角色
   - 当前排班状态

3. **排班类别**
   - 上午班（08:00-12:00）
   - 下午班（13:00-17:00）
   - 夜班（17:00-次日08:00）
   - 休息

### 子系统三：检查管理子系统

#### 3.4 工作列表页面（/worklist）

**页面路径**：首页 → 工作列表

**页面功能**：工作列表是核医学科技师和护士的每日工作入口。系统根据当日预约自动生成检查工作清单，支持按检查类型、设备、状态等多维度筛选，帮助工作人员高效完成每日检查任务。

**主要功能**：

1. **今日检查列表**
   - 自动显示当日所有预约检查
   - 按检查时间排序
   - 实时更新检查状态

2. **状态筛选**
   - 待执行：等待执行的检查
   - 执行中：正在进行的检查
   - 已完成：已完成采集的检查
   - 全部：显示所有

3. **类型筛选**
   - PET/CT检查
   - SPECT检查
   - 核素治疗
   - 骨密度检查

4. **快速操作**
   - 刷新列表
   - 标记执行中
   - 标记已完成

**检查状态说明**：

| 状态 | 说明 | 颜色标识 |
|------|------|----------|
| 准备中 | 患者准备中（更衣、禁食等） | 蓝色 |
| 检查中 | 正在进行图像采集 | 黄色 |
| 图像采集中 | PET/CT或SPECT采集中 | 黄色 |
| 处理中 | 图像后处理中 | 蓝色 |
| 已完成 | 检查完成，等待报告 | 绿色 |

#### 3.5 检查记录页面（/exams）

**页面路径**：首页 → 检查记录

**页面功能**：检查记录页面展示所有历史检查的详细信息，包括检查参数设置、图像采集数据、SUV值计算结果、影像所见描述等，是核医学检查数据管理的核心模块。

**主要功能**：

1. **检查记录查询**
   - 按患者姓名、检查类型、日期范围查询
   - 支持检查状态筛选

2. **PET/CT检查详情**
   - 检查基本信息：设备型号、采集时间、图像数量
   - SUV参数：SUVmax、SUVpeak、肝脏SUV背景值
   - 图像系列数
   - 图像所见（findings）
   - 诊断结论（conclusion）

3. **SPECT检查详情**
   - 检查项目类型
   - 注射剂量和时间
   - 采集参数
   - GFR值（肾动态显像）
   - SSS/SRS评分（心肌灌注显像）
   - 图像所见和结论

4. **骨密度检查详情**
   - BMD_L1_L4：腰椎L1-L4骨密度值
   - BMD_FemoralNeck：股骨颈骨密度值
   - T-Score：T评分（骨质疏松诊断标准）

**数据字段说明**：

| examItem | 检查项目 | 全身肿瘤筛查 |
| examDate | 检查日期 | 2026-05-02 |
| startTime | 开始时间 | 08:30 |
| endTime | 结束时间 | 09:45 |
| status | 检查状态 | 已完成 |
| device | 检查设备 | PET/CT-01 |
| doctor | 主诊医生 | 张明华 |
| nurse | 护士 | 陈护士 |
| radiopharmaceutical | 放射性药物 | 18F-FDG |
| injectedDose | 注射剂量 | 5.5mCi |
| SUVmax | 最大标准摄取值 | 8.2 |
| findings | 图像所见 | 右肺上叶见一枚代谢增高结节 |
| conclusion | 诊断结论 | 右肺上叶恶性病变可能性大 |

**图像采集操作流程**：
1. 工作人员在工作列表中找到当日检查任务
2. 点击进入检查记录页面
3. 核对患者信息和检查项目
4. 执行检查前准备（患者更衣、确认禁食等）
5. 注射放射性药物
6. 到达等待时间后开始图像采集
7. 采集过程中记录关键参数
8. 采集完成后进行图像后处理
9. 填写图像所见和诊断结论
10. 提交检查记录

### 子系统四：报告管理子系统

#### 3.6 报告管理页面（/reports）

**页面路径**：首页 → 报告管理

**页面功能**：报告管理页面提供检查报告的查询、审核、发布全流程管理功能。系统支持报告状态追踪、报告模板调用、报告批量导出等操作，是核医学科报告质量控制的重要工具。

**主要功能**：

1. **报告查询**
   - 按患者姓名、检查类型、报告日期查询
   - 支持报告状态筛选

2. **报告状态流转**
   - 待书写：新建报告，等待书写
   - 待审核：报告已书写，等待审核
   - 已审核：报告已审核，等待发布
   - 已发布：报告已发布给患者/临床

3. **报告详情查看**
   - 基本信息：报告编号、患者姓名、检查类型
   - 检查信息：检查日期、设备、技术参数
   - 报告内容：临床指征、检查技术、图像所见、诊断结论
   - SUV参数（如适用）
   - 报告医师和审核医师签名

4. **报告发布**
   - 审核通过后点击"发布"按钮
   - 支持打印报告
   - 支持电子签名

**报告模板说明**：

系统为不同检查类型预置了标准化报告模板：


#### 3.7 报告书写页面（/report-write）

**页面路径**：首页 → 报告书写

**页面功能**：报告书写页面是核医学科医生书写诊断报告的工作平台。系统提供结构化报告模板，支持SUV值自动提取、图像数量统计、常用语快捷插入等功能，大幅提升报告书写效率。

**主要功能**：

1. **患者信息自动加载**
   - 从检查记录自动获取患者基本信息
   - 从检查设备自动获取采集参数

2. **报告内容编辑**
   - 临床指征：下拉选择或手动输入
   - 检查技术：自动填充，可修改
   - 图像所见：富文本编辑，支持常用语
   - 诊断结论：结构化输入，支持常用结论模板

3. **SUV参数管理（PET/CT）**
   - 自动显示本次检查SUVmax、SUVpeak
   - 显示肝脏SUV背景值
   - 支持与历史检查SUV值对比

4. **图像数量统计**
   - 自动统计PET/CT采集帧数和系列数
   - 自动统计SPECT图像数量

5. **报告保存与提交**
   - 草稿保存：临时保存报告，不提交审核
   - 提交审核：完成书写，提交上级医师审核
   - 一键发布（需审核权限）：直接发布报告

**报告书写操作流程**：
1. 从报告列表点击"新建报告"或选择已有检查记录
2. 系统自动加载患者信息和检查参数
3. 选择或输入临床指征
4. 核对检查技术描述
5. 书写图像所见（可调用常用语模板）
6. 书写诊断结论（可调用常用结论模板）
7. 检查SUV参数（如适用）
8. 点击"保存"或"提交审核"
9. 如需审核，等待上级医师审核通过
10. 审核通过后，报告状态变更为"已发布"

## 4. 核心业务流程

### 4.1 患者预约→检查→报告流程

这是核医学科最核心的诊疗流程，全流程如下：

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  预约登记   │───▶│  预约确认   │───▶│  患者到检   │───▶│  检查执行   │───▶│  报告书写   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
                                                                                   │
                                                                                   ▼
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   归档存储   │◀───│  报告发布   │◀───│  报告审核   │◀──│  报告提交   │◀──│  图像采集   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

**详细步骤说明**：

#### 第一步：预约登记（预约管理页面）

1. 医生或护士在预约管理页面点击"新建预约"
2. 选择患者（从患者库选择或新建患者档案）
3. 选择检查类型（PET/CT/SPECT/核素治疗/骨密度）
4. 选择具体检查项目
5. 选择预约日期和时间
6. 选择主诊医生
7. 如有特殊要求（如糖尿病史、禁食要求），在备注中说明
8. 保存预约信息

**预约成功提示**：
- 系统显示预约编号
- 自动发送预约提醒（短信/APP推送）
- 预约状态显示为"待确认"

#### 第二步：预约确认

1. 主诊医生确认预约信息
2. 如需要特殊准备（如PET/CT检查前禁食6小时），提前通知患者
3. 确认放射性药品需求
4. 点击"确认预约"按钮
5. 预约状态变更为"已确认"

#### 第三步：患者到检（工作列表页面）

1. 患者按预约时间到达核医学科
2. 护士在工作列表中找到该患者
3. 核对患者身份（姓名、身份证号）
4. 确认检查项目和注意事项
5. 如为PET/CT检查，测量空腹血糖（需控制在8.0mmol/L以下）
6. 引导患者进入准备区

#### 第四步：检查执行（检查记录页面）

1. **注射前准备**
   - 护士从药品分发页面领取放射性药品
   - 核对药品名称、批号、剂量
   - 在药品分发页面新建分发记录

2. **注射放射性药物**
   - 严格执行"三查七对"
   - 按处方剂量准确注射
   - 记录注射时间

3. **等待期**
   - 18F-FDG：等待45-60分钟
   - 99mTc标记物：等待15-30分钟
   - 患者在休息区安静等待

4. **图像采集**
   - 技师引导患者进入检查室
   - 根据检查项目调整患者体位
   - 启动设备进行图像采集
   - PET/CT采集时间约15-25分钟
   - SPECT采集时间约20-30分钟

5. **图像后处理**
   - 技师进行图像重建
   - 计算SUV值（如适用）
   - 传输图像到PACS系统

6. **记录检查信息**
   - 在检查记录页面完善检查信息
   - 填写图像所见
   - 书写初步结论

#### 第五步：报告书写（报告书写页面）

1. 主诊医生登录系统
2. 从报告管理页面找到待书写报告
3. 选择对应检查记录
4. 系统自动加载患者信息和检查参数
5. 书写或确认临床指征
6. 核对检查技术描述
7. 书写图像所见（可调用常用语模板）
8. 书写诊断结论（可调用常用结论模板）
9. 检查SUV参数是否正确
10. 点击"保存"或"提交审核"

#### 第六步：报告审核

1. 上级医师登录系统
2. 从报告列表中找到待审核报告
3. 仔细阅读报告内容
4. 如有疑问，可打回给报告医师修改
5. 如报告无误，点击"审核通过"
6. 报告状态变更为"已审核"

#### 第七步：报告发布

1. 审核通过后，点击"发布报告"
2. 系统自动生成最终报告
3. 支持打印报告
4. 报告自动归档到患者档案
5. 临床科室可在线查阅报告

#### 第八步：归档存储

1. 检查图像自动归档到PACS系统
2. 报告归档到RIS/HIS系统
3. 药品使用记录归档
4. 所有数据支持长期存储和检索

### 4.2 药品管理流程

放射性药品管理是核医学科管理的重中之重。系统实现药品全生命周期管理：

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   药品入库   │───▶│   药品存储   │───▶│   药品分装   │───▶│  药品注射   │───▶│  残余处理   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

**详细步骤说明**：

#### 第一步：药品入库

1. 供应商送达放射性药品
2. 库管员核对：
   - 药品名称、批号
   - 活度（mCi）
   - 效期
   - 供应商资质
   - 药品批准文号
3. 在药品库存页面新增药品入库记录
4. 按要求存储（冷藏/-20°C冷冻/铅罐屏蔽）

#### 第二步：药品存储

1. **存储条件要求**：

| 药品类型 | 存储条件 | 注意事项 |
|----------|----------|----------|
| 18F-FDG | -20°C冷冻 | 避免反复冻融 |
| 99mTc标记物 | 室温保存 | 避免高温 |
| 131I-NaI | 铅罐屏蔽，通风保存 | 防辐射 |
| 89SrCl2 | 室温保存 | 按要求屏蔽 |

2. 系统自动监控效期
3. 临近效期（3天内）自动预警

#### 第三步：药品分装

1. 护士根据处方取出药品
2. 使用经校准的活度计测量活度
3. 按处方剂量分装到注射器
4. 记录分装时间、分装人、分装剂量
5. 在药品分发页面更新分发状态为"已分装"

#### 第四步：药品注射

1. 注射前再次核对：
   - 患者姓名、身份证号
   - 药品名称、剂量
   - 注射部位
2. 规范注射操作
3. 记录注射时间
4. 注射器及残余药品按规定处理
5. 更新药品分发状态为"已注射"

#### 第五步：残余处理

1. 测量残余活度
2. 记录残余量
3. 按放射性废物处理规程处理：
   - 短半衰期废物（99mTc）：衰变存储10个半衰期后按普通废物处理
   - 长半衰期废物（131I）：专用放射性废物容器存储
   - 液体废物：衰变池存储
4. 在药品分发页面记录废弃处理方式
5. 更新分发状态为"已废弃"

### 4.3 辐射防护流程

核医学科辐射防护管理是保护工作人员职业健康的重要措施：

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  剂量计佩戴 │───▶│  日常操作   │───▶│  月度检测   │───▶│  数据录入   │───▶│  阈值监控   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

**详细步骤说明**：

#### 第一步：剂量计佩戴

1. 工作人员上岗前领取个人剂量计
2. 正确佩戴：
   - 铅玻璃剂量计：佩戴于左胸前（甲状腺防护内侧）
   - 指环剂量计：操作放射性药物时佩戴于食指
3. 注意：
   - 不得擅自拆换
   - 不得带入辐射区外
   - 远离热源和磁场

#### 第二步：日常操作规范

1. **注射操作**：
   - 使用铅注射器防护屏
   - 佩戴铅手套
   - 操作后立即盖上铅罐

2. **给药操作**：
   - 使用长柄给药器
   - 保持最大操作距离
   - 减少操作时间

3. **废物处理**：
   - 使用铅废物罐
   - 及时封存废物

4. **环境监测**：
   - 每日检测工作台面污染
   - 每周检测机房辐射水平

#### 第三步：月度检测

1. 月底将剂量计送交职业病防治院检测
2. 检测项目：
   - 深部剂量Hp(10)
   - 浅表剂量Hp(0.07)
   - 肢体剂量Hp(0.07)
3. 获取检测报告

#### 第四步：数据录入

1. 物理师登录系统
2. 在辐射防护页面新建剂量记录
3. 录入当月各剂量值
4. 系统自动计算年累计剂量
5. 系统自动判定状态（正常/关注/超标）

#### 第五步：阈值监控与预警

1. **年度剂量阈值**：
   - 职业照射：50mSv/年
   - 公众照射：20mSv/年

2. **月度关注阈值**：
   - 5mSv/月（可调）

3. **预警机制**：
   - 月剂量超5mSv：系统显示"关注"
   - 年累计超40mSv（80%阈值）：系统发出预警
   - 年累计超50mSv：系统显示"超标"，通知科室负责人

4. **超标处理**：
   - 立即复测确认
   - 分析超标原因
   - 调整工作岗位
   - 安排医学观察

## 7. 常见问题

### 7.1 系统登录问题

#### Q1：忘记密码怎么办？
A：请联系系统管理员（李主任，工号U007），管理员可为您重置密码。重置后首次登录需修改密码。

#### Q2：登录提示"账号或密码错误"？
A：请检查：
1. 确认工号输入正确（大写U开头，如U001）
2. 确认密码大小写正确
3. 确认Caps Lock未开启
4. 如连续3次输入错误，账号将被锁定15分钟

#### Q3：登录后页面显示异常？
A：请尝试：
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 强制刷新页面（Ctrl+F5）
3. 更换浏览器（推荐Chrome）
4. 如问题依旧，请联系技术支持

### 7.2 预约管理问题

#### Q4：无法新建预约，提示"患者信息不存在"？
A：请先在患者管理页面新增患者档案，再进行预约登记。

#### Q5：预约时间无法选择？
A：请检查：
1. 确认选择的日期不是过去日期
2. 确认当日预约名额未满
3. 确认设备在该时段可用

#### Q6：预约需要取消怎么办？
A：在预约管理页面找到目标预约，点击"取消"按钮，选择取消原因即可。取消后的预约号不再使用。

### 7.3 检查执行问题

#### Q7：检查记录无法保存？
A：请检查：
1. 必填字段是否完整
2. SUV值格式是否正确（数字格式）
3. 检查时间逻辑是否正确（结束时间晚于开始时间）

#### Q8：PET/CT检查血糖超标怎么办？
A：根据规范，PET/CT检查前空腹血糖应控制在8.0mmol/L以下。如超标：
1. 轻度超标（8.0-10.0mmol/L）：可正常检查，但需在报告中注明
2. 重度超标（>10.0mmol/L）：建议调整血糖后再行检查

#### Q9：药品注射后发现注射部位有渗漏？
A：如发现注射渗漏：
1. 记录渗漏情况
2. 在报告备注中说明
3. 计算实际注入剂量
4. 报告审核时提示核医学科医生

### 7.4 报告管理问题

#### Q10：报告书写时无法调取检查图像？
A：请检查：
1. 确认检查已完成且状态为"已完成"
2. 确认图像已成功上传到PACS系统
3. 联系技术支持检查PACS连接状态

#### Q11：报告提交审核被退回？
A：如报告被退回：
1. 查看审核意见
2. 按意见修改报告内容
3. 重新提交审核

#### Q12：报告审核通过后可以修改吗？
A：报告一旦审核通过并发布，修改需经过以下流程：
1. 联系审核医师申请解锁
2. 解锁后可修改内容
3. 重新提交审核
4. 发布新版本报告

### 7.5 药品管理问题

#### Q13：药品库存显示负数怎么办？
A：如发现库存为负数：
1. 检查是否有未登记的出库记录
2. 检查分发记录数据是否准确
3. 如确认数据无误，请联系管理员进行库存校正

#### Q14：药品临近过期如何处理？
A：根据药品效期管理规范：
1. 有效期前3天：系统预警，优先使用
2. 过期当天：停止使用，下架封存
3. 过期后：按规定进行废弃处理

#### Q15：药品分发记录填错了怎么办？
A：在药品分发页面找到错误记录，点击"编辑"按钮修改。如记录已归档，需联系管理员解锁后修改。

### 7.6 辐射防护问题

#### Q16：个人剂量超标后怎么办？
A：如发现个人剂量超阈值：
1. 立即停止放射性操作
2. 联系科室负责人和职业病管理人员
3. 复查剂量计确认数据
4. 安排医学健康检查
5. 评估是否需要调整工作岗位

#### Q17：剂量计丢失怎么办？
A：如剂量计丢失：
1. 立即报告科室负责人
2. 补发新剂量计
3. 如在丢失期间可能接受较大剂量，需进行评估
4. 填写剂量计丢失报告

### 7.7 质控管理问题

#### Q18：设备质控不合格怎么办？
A：如设备质控不合格：
1. 停止使用该设备进行临床检查
2. 联系设备维修工程师
3. 进行设备维修和校准
4. 维修后重新进行质控检测
5. 质控合格后方可恢复临床使用

#### Q19：质控记录可以修改吗？
A：历史质控记录原则不可修改，以确保数据可追溯性。如发现记录错误，请新建一条更正记录，并备注说明。

### 7.8 其他问题

#### Q20：系统运行缓慢怎么办？
A：如系统响应慢：
1. 检查网络连接是否稳定
2. 清除浏览器缓存
3. 关闭不必要的后台程序
4. 如问题持续，请联系技术支持

#### Q21：如何导出数据报告？
A：在统计报表页面，选择要导出的数据范围，点击"导出"按钮，系统将生成Excel格式的数据文件。

#### Q22：系统数据如何备份？
A：系统支持自动备份功能：
1. 每日凌晨3点自动备份
2. 备份数据保留30天
3. 如需人工备份，请联系系统管理员

#### Q23：培训记录如何录入？
A：在教育培训页面，选择"新建培训记录"，填写培训主题、培训时间、参训人员等信息，上传相关培训材料即可。

#### Q24：用户权限如何申请？
A：用户权限申请流程：
1. 填写权限申请表（说明需要哪些权限）
2. 科室负责人签字确认
3. 提交至系统管理员
4. 管理员审核后配置权限
